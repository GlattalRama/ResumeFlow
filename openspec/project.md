# ResumeFlow Project

ResumeFlow is a resume builder and job application tracking portal.

Users can:
- create resumes from JSON data
- preview resumes in multiple templates
- save different resume versions
- link each resume version to job applications
- store job descriptions and job IDs
- track application status
- add notes
- prepare interview Q&A

## MVP Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- JSON files as backend storage
- No database
- No authentication
- No real AI API yet

## Main Features

- Resume builder
- Multiple resume templates
- Resume versioning
- Job application tracking
- Notes
- Interview Q&A
- Status history
- Document metadata
- AI placeholder buttons
